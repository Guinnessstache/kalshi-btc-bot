FULLY AUTOMATED
Monitors Kalshi's BTC 15-minute markets continuously. Enters YES positions when price crosses your trigger, manages the trade, and exits automatically.
📈
TRAILING STOP
As price rises above entry, the stop-loss trails up — locking in profit automatically. If price reverses, you exit with gains instead of a loss.

VELOCITY EXIT
Detects fast crashes before slippage gets worse. If price drops more than X¢ in a single half-second poll, the bot exits immediately.

STALL EXIT
If price hasn't moved above your entry after N seconds, the bot cuts the trade early — eliminating the slow bleed before it becomes a big loss.

GETTING STARTED

Up and Running
in 4 Steps

No admin rights required. The build script handles Python automatically.

01
DOWNLOAD & EXTRACT
Grab the ZIP from this page and extract it to any folder on your Windows PC. Run it from anywhere — no installation, no admin rights needed.
KalshiBTCBot.zip → extract anywhere
02
BUILD THE EXE
Double-click build_windows.bat. It installs Python dependencies and compiles a standalone KalshiBot.exe. Takes about 60 seconds.
build_windows.bat → dist/KalshiBot.exe
03
ENTER YOUR API KEY
Launch KalshiBot.exe. Enter your Kalshi API Key ID and browse to your private key file. Find these at kalshi.com → Settings → API. Paper mode lets you test without real money.
kalshi.com → Settings → API → Generate Key
04
WATCH IT TRADE
The dashboard opens automatically. Monitor live BTC/USD price, session P&L, trade history, and the activity log — all in real time.
http://localhost:5050
