Everything You Need to
Trade Smarter

Built on Kalshi's official API — fully automated, fully configurable, no cloud required.

⚡
FULLY AUTOMATED
Monitors Kalshi's BTC 15-minute markets continuously. Enters YES or NO positions when price crosses your trigger, manages the trade, and exits automatically.
📈
TRAILING STOP
As price rises above entry, the stop-loss trails up — locking in profit automatically. If price reverses, you exit with gains instead of a loss.
🚀
VELOCITY EXIT
Detects fast crashes before slippage gets worse. If price drops more than X¢ in a single half-second poll, the bot exits immediately.
⏱
STALL EXIT
If price hasn't moved above your entry after N seconds, the bot cuts the trade early — eliminating the slow bleed before it becomes a big loss.
NEW
🔻
NO ONLY MODE
Trade both directions. Switch to NO-only during bearish BTC conditions — the bot enters NO positions when NO price crosses your trigger. Pairs with Hold to Expiry for maximum edge.
NEW
📌
HOLD TO EXPIRY
Disable all stop-losses and let every position ride to window close. Perfect for NO-only mode — prevents premature exits during BTC bounces that ultimately resolve in your favour.
NEW
📊
VOLUME FILTER
Skip thin markets automatically. Set a minimum contract volume threshold — the bot only enters markets with enough liquidity to avoid order book slippage.
🖥
LIVE DASHBOARD
Real-time BTC/USD chart, trade history, reconcile tab, session P&L, win rate, and activity log — all in a browser at localhost:5050.
🔒
RUNS LOCALLY
Everything runs on your Windows PC. No cloud, no subscriptions, no third parties. Your Kalshi API keys never leave your machine.

GETTING STARTED

Up and Running
in 4 Steps

No admin rights required. The build script handles Python automatically.

01
DOWNLOAD & EXTRACT
Grab the ZIP from this page and extract it to any folder on your Windows PC. Inside you'll find all source files and build_windows.bat to compile the app.
KalshiBTCBot.zip → extract all files
02
BUILD THE EXE
Double-click build_windows.bat. It installs Python if needed, bundles everything, and outputs dist\KalshiBot.exe — a fully standalone app with no dependencies.
build_windows.bat → dist\KalshiBot.exe
03
CONFIGURE & LAUNCH
Run KalshiBot.exe. The setup window opens — choose Live or Paper mode, enter your Kalshi API key, set your trade size, and pick your trading direction (YES only, NO only, or both).
kalshi.com → Settings → API → Generate Key
04
WATCH IT TRADE
The dashboard opens automatically in your browser. Monitor live BTC/USD price, session P&L, trade history, reconcile tab, and the activity log — all in real time.
http://localhost:5050
